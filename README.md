# CV_Project_PlantVillage
Developed a plant disease classification system using the PlantVillage dataset (54K images, 38 classes). Compared HOG+ORB+SVM, a custom CNN, augmented CNN, and ResNet50 transfer learning. ResNet50 achieved the best accuracy (97.73%) and robustness to noise and distortions, while adversarial testing revealed deep learning vulnerabilities.
